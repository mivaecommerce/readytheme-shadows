Additional Notes

While every effort has been made to make sure this ReadyTheme performs across a variety of devices, we know that the current use of @import in the CSS files can cause some site speed services to rank the site slower. If you would prefer to combine all the CSS files into one, you can use the following steps. This code is included in the default installation of this ReadyTheme.

  1. Navigate to ReadyTheme -> Content Sections and activate ReadyTheme Styles.
  2. Navigate to User Interface -> CSS Resources and deactivate site-styles.

If you add more style sheets or extension to your site, you will need to update the list of files to be concatenated in ReadyTheme Styles.
